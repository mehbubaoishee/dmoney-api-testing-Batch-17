# Dmoney REST API Automation Testing

**Project on Postman & Newman**  
**Name:** [Mehbuba Alam Oishee]  

---

### Project Description
This project automates testing of the Dmoney REST API using Postman and Newman.  
It covers 10 main scenarios (Admin → Agent → Customer → Merchant transactions) with positive and negative test cases, request chaining, test scripts, and full reporting.

### Technology & Tools Used
- Postman (Collection + Tests + Documentation)
- Newman + newman-reporter-htmlextra
- Node.js & npm
- Git & GitHub
- Google Sheets (Test Cases & Bug Report)

### How to Run the Project
- Clone the repo
- Install dependencies
- npm install
- Run tests and generate HTML report
